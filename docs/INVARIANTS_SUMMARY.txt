## Infrastructure Invariants

**Repository Structure:**
- Exactly one `etc/` directory at `GH-etc/etc/` (nginx + systemd configs)
- Exactly one `srv/webapps/` skeleton at `GH-etc/srv/webapps/` (no runtime artifacts)
- No `.git` directories except the root repository
- No `venv/`, `__pycache__/`, or compiled Python files

**Path References:**
- All nginx and systemd configs reference system paths (`/srv/webapps/...`, `/etc/...`)
- No references to home directory paths (`/home/admin/etc`, `/home/admin/srv`)

**Deployment Safety:**
- GH-etc is the single canonical source of truth
- Live system paths (`/etc`, `/srv/webapps`) are never modified directly
- Deployment scripts sync from GH-etc to live system paths only

**Verification:** See `docs/INFRASTRUCTURE_INVARIANTS.md` for detailed verification commands and procedures.
